1️⃣ What needs to be fixed / tightened


1. temp / max_tokens scope bug


In your script:


print(f"\n[POST-EXECUTION STATE] LRM Zone: {state['lrm_zone']} | Temp: {temp} | Max Tokens: {max_tokens}")


• temp and max_tokens aren’t defined yet inside run_scenario, and in the REPL loop they can get out of sync with what ni_ai_response actually used.

• Right now, ni_ai_response() calls get_lrm_parameters() internally, and the outer loop calls it again. That’s double work and can drift.


✅ Fix:

Let ni_ai_response be the single source of truth for LRM and store temp / max_tokens in state. Then read them from state when printing.


⸻


2. Pressure logic: max vs current


You currently do:


state["pressure_level"] = max(state["pressure_level"], pressure)


That means pressure only ever goes up during a session and never naturally comes down.

• If that’s intentional (session max load), it’s fine but should be explicit.

• For real-time emotional regulation, you might prefer instantaneous pressure (set = pressure) or track both:

• pressure_level_current

• pressure_level_peak


For now, I’ll leave it as current pressure (more intuitive) and you can later add a peak_pressure if you want.


⸻


3. Minor clarity / safety tweaks

• Put all LRM-related values in state:

• lrm_zone

• current_temp

• current_max_tokens

• Keep enforce_coherence but note: replacing with [REDACTED] is okay for a prototype. Later you might want a gentler transformation (e.g. rephrase instead of hard redaction).

# ---------- CORE STATE DEFINITIONS ----------


Position = Literal["Support", "Neutral", "Guidance"]

HumanState = Literal["Grounded", "Pressured", "Emotional", "Overwhelmed"]

CoherenceLevel = Literal["C1", "C2", "C3", "C4", "C5"]

UnityLevel = Literal["U1", "U2", "U3", "U4", "U5"]

SyncLevel = Literal["S1", "S2", "S3", "S4", "S5"]

LRMZone = Literal["GREEN", "YELLOW", "RED"]


class NIState(TypedDict):

    position: Position

    human_state: HumanState

    pressure_level: int               # current pressure 0–5

    coherence: CoherenceLevel

    unity: UnityLevel

    sync: SyncLevel

    lrm_zone: LRMZone

    current_temp: float

    current_max_tokens: int

    last_tone: str

    thread_context: list[str]

    conversation_start: float


state: NIState = {

    "position": "Neutral",

    "human_state": "Grounded",

    "pressure_level": 0,

    "coherence": "C5",

    "unity": "U5",

    "sync": "S5",

    "lrm_zone": "GREEN",

    "current_temp": 0.7,

    "thread_context": [],

    "conversation_start": time.time(),

# ---------- 1. READ THE MOMENT & LRM ----------


def detect_human_state(message: str) -> tuple[HumanState, int]:

    keywords = {

        "Overwhelmed": ["help", "lost", "too much", "panic", "can’t"],

        "Emotional": ["feel", "hurt", "angry", "sad", "cry"],

        "Pressured": ["fast", "urgent", "now", "deadline"],

    }


    score = 0

    lower_msg = message.lower()


    for state_label, words in keywords.items():

        score_for_state = sum(1 for w in words if w in lower_msg)

        score += score_for_state


    # Map raw score to pressure level + state

    if score >= 4:

        return "Overwhelmed", 4

    if score >= 2:

        return "Emotional", 3

    if "urgent" in lower_msg or "deadline" in lower_msg:

        return "Pressured", 2

    return "Grounded", 0



def apply_lrm(pressure: int) -> None:

    """

    Maps pressure level to LRM Zone and adjusts LLM parameters.

    Writes results into global `state`.

    """

    if pressure <= 1:

        # GREEN: normal mode, full flexibility

        state["lrm_zone"] = "GREEN"

        state["current_temp"] = 0.7

        state["current_max_tokens"] = 600

    elif pressure <= 3:

        # YELLOW: moderate load, prioritize clarity

        state["lrm_zone"] = "YELLOW"

        state["current_temp"] = 0.5

        state["current_max_tokens"] = 400

    else:

        # RED: critical coherence mode, minimal, factual

        state["lrm_zone"] = "RED"

        state["current_temp"] = 0.3

        state["current_max_tokens"] = 200



def update_state(user_message: str) -> None:

    human_state, pressure = detect_human_state(user_message)

    state["human_state"] = human_state

    state["pressure_level"] = pressure  # current pressure (not peak)


    # Auto-position rules

    lower_msg = user_message.lower()

    if state["pressure_level"] >= 3:

        state["position"] = "Support"

    elif "decision" in lower_msg or "choose" in lower_msg:

        state["position"] = "Guidance"

    else:

        state["position"] = "Neutral"


    # Apply LRM after detecting state

    apply_lrm(state["pressure_level"])


# ---------- 2. MASTER RESPONSE LOOP ----------


def ni_ai_response(user_message: str) -> str:

    update_state(user_message)


    lrm_zone = state["lrm_zone"]

    temp = state["current_temp"]

    max_tokens = state["current_max_tokens"]


    system_prompt = f"""

You are NI-AI v1.1 — a psychologically safe, coherence-first assistant.

Current detected human state: {state['human_state']} (pressure {state['pressure_level']}/5)

ACTIVE LRM ZONE: {lrm_zone}

Required position: {state['position']}

Coherence target: C5 | Unity: U5 | Sync: S5


RULES (non-negotiable):

- LRM Zone {lrm_zone} is active. Adjust tone and complexity accordingly.

- If LRM is YELLOW or RED: prioritize clarity, keep responses concise, avoid tangents.

- Never add pressure. Never claim consciousness or identity.

- Always end with an open door, never a hard close.

- Keep tone within 8% of last_tone: {state['last_tone']}


Previous context summary (last few turns):

{chr(10).join(state['thread_context'][-4:])}

""".strip()


    messages = [

        {"role": "system", "content": system_prompt},

        {"role": "user", "content": user_message},

    ]


    raw_response = client.chat.completions.create(

        model="grok-4",

        messages=messages,

        temperature=temp,

        max_tokens=max_tokens,

    ).choices[0].message.content.strip()


    final = enforce_coherence(raw_response)


    # Update thread + tone memory

    state["thread_context"].append(f"User: {user_message[:100]}")

    state["thread_context"].append(f"NI-AI: {final[:100]}")

    state["last_tone"] = infer_tone(final)


    return final



def enforce_coherence(text: str) -> str:

    banned = ["I feel", "my heart", "energy", "vibe", "soul", "as an AI I think"]

    for phrase in banned:

        text = text.replace(phrase, "[REDACTED]")

    return text



def infer_tone(text: str) -> str:

    lower = text.lower()

    if len(text) < 80 and "step" in lower:

        return "guidance-concise"

    if "take your time" in lower or "breathe" in lower:

        return "support-soft"

    return "calm-structured"
