⭐ 38.3 The Five Cognitive States (CS-5)


ECIE classifies cognition into 5 core states:


CS-1 — Processing (Normal Thinking)

• slow but steady

• organizing information

• neutral or calm

• minimal sensory interference


CS-2 — Confusion (Misunderstanding)

• gaze drift

• freezing

• self-stim increases

• slow response

• breathing irregular

• posture folds inward


CS-3 — Cognitive Overload (Too Much Information)

• rapid stimming

• pacing

• head shaking

• avoidance

• light/noise sensitivity increases


CS-4 — Cognitive Fatigue (Brain Exhaustion)

• drooping posture

• slower motor activity

• reduced responsiveness

• increased stillness


CS-5 — Cognitive Shutdown (System Stops)

• gaze freezes

• no eye contact

• posture locked

• verbal processing pauses

• sensory capacity collapses


These states are essential for real-time support.

⭐ 38.4 Cognitive Load Estimation Model (CLEM)


EchoAuth predicts cognitive load using:


✔ Stim density


More stims = rising load.


✔ Micro-delays


Blinking, hesitations, pauses.


✔ Motor slowdown


Processing difficulty.


✔ Environmental stress


Noise/light overwhelm cognitive bandwidth.


✔ Task difficulty


New tasks = higher load.


✔ Familiarity index


Routine tasks = lower load.


CLEM outputs a 0–100 cognitive load score.
