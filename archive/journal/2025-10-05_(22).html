README.md

# EchoAuth Chamber


A production-ready biometric verification system for real-time face and voice authentication using PyTorch with CUDA optimizations (CUDA graphs, FP16 AMP, channels_last). Supports multi-threaded input feeds, micro-batching, and latency monitoring.


## Requirements

- Python 3.8+

- NVIDIA GPU (e.g., RTX 3080) with CUDA 11.8+ for GPU acceleration

- Dependencies: `pip install -r requirements.txt`


## Files

- `echoauth_chamber.py`: Main verification pipeline with thread-safe `submit_sample` and `get_latest_result` APIs.

- `export_onnx.py`: Exports models to ONNX for TensorRT conversion.

- `build_trt_module.py`: Compiles to Torch-TensorRT module (optional, requires `torch-tensorrt`).

- `capture_feed.py`: Captures webcam/mic data and feeds to `submit_sample` (requires `opencv-python`, `pyaudio`).
