2️⃣ Caregiver & Regulator Annotations


(Boundary Callouts — Stored with Diagram)


These annotations surround the diagram.

They are not embedded inside the boxes.


Governance Boundaries

• EchoAuth is not diagnostic

• EchoAuth is not therapeutic

• EchoAuth does not analyze behavior

• EchoAuth is an authentication and access-control interface


Authority Model

• Parent or legal guardian defines all rules

• Caregivers execute only within approved roles

• No caregiver overrides

• No silent rule changes


Safety & Audit

• If uncertainty exists → access pauses

• Every access attempt is logged

• Logs include timestamp, role, outcome

• Audit trail is immutable
