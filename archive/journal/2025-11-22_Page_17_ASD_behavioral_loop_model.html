PAGE 17 — ASD BEHAVIORAL LOOP MODEL (ABLM)


“Behavior is a loop, not an event.”


──────────────────────────────────


Autistic behavior doesn’t appear randomly.

It forms predictable loops based on:

• sensory load

• emotional spikes

• cognitive blocking

• environmental triggers

• transitions

• communication difficulty


When EchoAuth understands the loop, it can break escalation before it starts.


──────────────────────────────────


⭐ 17.1 — THE 5 BEHAVIORAL LOOP PHASES


Every ASD behavior cycle moves through these stages:


PHASE 1 — Trigger Signal


Something changes in the environment:

• sound

• light

• movement

• expectation

• instruction

• people

• temperature

• pace


EchoAuth must detect the first micro-shift

→ a drop in regulation

→ a hesitation

→ a pause


This is the most important phase.


⸻


PHASE 2 — Sensory-Emotional Surge


The nervous system spikes faster than neurotypical profiles.


Signs:

• fidgeting

• pressure in the chest

• eye avoidance

• pacing

• verbal repetition

• sudden quietness


EchoAuth must shift to Safe Mode here if the spike reaches threshold.


⸻


PHASE 3 — Cognitive Disruption


Thought processing becomes harder.


Not “slower.”

Just blocked.


Signs:

• confusion

• repeating the same question

• silence

• trouble choosing

• avoiding decisions

• repeating a phrase


EchoAuth must switch to:

• grounding language

• no choices

• no new topics

• 2–5 word sentences


⸻


PHASE 4 — Behavioral Expression


The nervous system releases the internal overload through:

• stimming

• freeze/shutdown

• crying

• pacing

• going quiet

• repetition

• irritability

• withdrawal


EchoAuth must:

• maintain predictable tone

• offer safe phrasing

• reduce cognitive load to minimum

• avoid emotional questions

• avoid problem-solving


⸻


PHASE 5 — Regulation / Reset Phase


Behavior stabilizes.


EchoAuth’s job here:

• restore safety

• slow pacing

• validate the experience

• offer choices ONLY if the user is ready


This prepares the system to return to Phase 1 baseline.

⭐ 17.2 — THE LOOP PATTERN (CYCLE MAP)


Trigger  

   ↓  

Sensory Surge  

   ↓  

Cognitive Disruption  

   ↓  

Behavior Expression  

   ↓  

Regulation / Reset  

   ↓  

(back to Trigger if environment unstable)


This loop shows where de-escalation is easiest:


⭐ EARLY STAGES (1–2) → Most effective for prevention

⭐ MID STAGE (3) → Requires grounding

⭐ LATE STAGE (4) → Requires Safe Mode only

⭐ RESET STAGE (5) → Best for rebuilding trust


EchoAuth is designed to intervene before Phase 4.

⭐ 17.4 — BEHAVIORAL LOOP CLASSIFICATIONS


There are four main loop types NI–AI must track.


⸻


1. Sensory-Dominant Loop


Triggered by:

• lights

• sound

• touch

• motion


NI–AI response:

→ Safe Mode

→ grounding

→ minimal speech


⸻


2. Emotional-Dominant Loop


Triggered by:

• fear

• pressure

• frustration

• confusion


NI–AI response:

→ slow pacing

→ literal language

→ reassurance


⸻


3. Cognitive-Dominant Loop


Triggered by:

• hard choices

• transitions

• unexpected instructions


Response:

→ step-by-step

→ predictable structure

→ visual pacing


⸻


4. Compound Loop


A blend of sensory + emotional + cognitive.


Most common.

Most challenging.


Response:

→ immediate multi-layer safe mode

→ no new information

→ hyper-predictability

⭐ 17.5 — HOW ECHOAUTH BREAKS THE LOOP


EchoAuth does not try to “fix behavior.”

It stabilizes the nervous system to prevent escalation.


Breakpoints:


Breakpoint A — Trigger Phase


→ “We can pause.”

→ “Slow and steady.”

→ “Here with you.”


Breakpoint B — Sensory Surge


→ “One breath.”

→ “You’re safe.”


Breakpoint C — Cognitive Block


→ “One step only.”

Breakpoint D — Behavioral Expression


→ “No rush.”

→ “I’m with you.”


EchoAuth always engages the lowest cognitive load response available.
