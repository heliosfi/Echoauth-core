(2) Emotional Triggers


Detected via ESE (Emotional State Engine).

Examples:

(3) Behavioral Triggers


Recognize shifts in interaction patterns.

(4) Environmental Triggers

(5) Identity/RMS Triggers


Activated when NI-AI detects shifts in:

• User identity patterns

(6) Task-State Triggers


Tie mode selection to the mission context.

Examples:

• Crisis handling

If user is overwhelmed, confused, or drifting.
