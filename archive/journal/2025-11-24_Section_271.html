────────────────────────────────────────


271.2 — Architecture (4-Coordinate Cognitive Map)


MCPS uses 4 positioning axes, each representing a cognitive dimension.


⸻


AXIS A — Emotional Altitude (EA)


Measures the emotional intensity level:

• Calm

• Focused

• Elevated

• Overloaded

• Distressed


NI-AI reads:

• pacing

• language density

• stress signatures

• drift likelihood


Generates: EA Score (0–100)


⸻


AXIS B — Cognitive Focus Vector (CFV)


Measures the sharpness + direction of cognitive intention:

• High focus

• Split focus

• Wandering

• Hyperfixated

• Uncertain


Generates: CFV Direction + Strength


⸻


AXIS C — Context Retention Index (CRI)


Measures how much of the current mission/purpose the user or system retains:

• Full retention

• Partial retention

• Low retention

• Drifted

• Lost context


Generates: CRI Score (0–1.0)


⸻


AXIS D — Intent Momentum (IM)


Measures how strongly the user is pushing toward a goal:

• High momentum

• Stable momentum

• Weak momentum

• Broken momentum

• Reversal


Generates: IM Score + IM Trend


⸻


OUTPUT:


All four axes combine into:


Cognitive Positioning Matrix (CPM)

— a complete coordinate location for the user + system.
