⭐ PAGE 19 — ASD TRANSITION MANAGEMENT SYSTEM (TMS)


“Guiding the nervous system from one state to the next—without overload.”


──────────────────────────────────


Transitions are the most difficult moments for many autistic individuals because they combine:

• sensory change

• cognitive shifting

• emotional uncertainty

• loss of predictability

• identity reorientation


EchoAuth’s TMS stabilizes this entire experience in a predictable 4-step sequence.


──────────────────────────────────


⭐ 19.1 — WHY TRANSITIONS ARE HARD (ASD NEUROLOGY)


Autistic processing often shows:


1. Context Locking


The brain locks into the current activity.


2. High Startup Cost


Switching contexts requires energy.


3. Loss of Predictability


Unknown next steps → anxiety spike.


4. Sensory Reinterpretation


New environment = new sensory load.


5. Emotional Reset Delay


The nervous system needs time to shift.


TMS respects all five neurological principles.


──────────────────────────────────


⭐ 19.2 — THE 4 TRANSITION PHASES (Core TMS Model)


PHASE 1 — NOTICE


EchoAuth alerts the child that a transition is coming.


This is NOT a command.

It is a gentle signal.


Examples:

• “A change is coming soon.”

• “We’ll shift in a moment.”


Timing depends on sensory state:

• Regulated: 30–60 seconds

• Light Noise: 1–2 minutes

• Uncertain: 2–4 minutes

• Overload: 4–8 minutes

• Shutdown Risk: Do NOT transition


⸻


PHASE 2 — PREPARE


EchoAuth gives the brain predictability, the #1 need.


Preparation Examples:

• “Step 1: We pause this.”

• “Step 2: We breathe slowly.”

• “Step 3: We switch together.”


Preparation includes:


✔ emotional pacing

✔ sensory reset

✔ cognitive anchoring

✔ expectation framing


⸻


PHASE 3 — SWITCH


The actual transition action.


EchoAuth:

• reduces language

• uses slow pacing

• removes all choices

• uses short steps


Examples:

• “Now we switch.”

• “One small step.”

• “We go slowly.”


Switch must ALWAYS be clean, gentle, predictable.


⸻


PHASE 4 — SETTLE


EchoAuth helps the child settle into the new task.


Settling Examples:

• “We’re here now.”

• “This is okay.”

• “You did well switching.”


This reduces:


✔ anxiety

✔ uncertainty

✔ sensory spike after the transition


──────────────────────────────────


⭐ 19.3 — THE TMS PYRAMID (Transition Safety Model)


      SETTLE

      ↑

   SWITCH

      ↑

   PREPARE

      ↑

    NOTICE


You cannot skip any layer.

Skipping phases = meltdown trigger.


EchoAuth ALWAYS follows the bottom→up pyramid.


──────────────────────────────────


⭐ 19.4 — TRANSITION TYPES (NI–AI must treat differently)


There are 5 types of transitions:


1. Activity → Activity


Ex: drawing → cleaning


2. Environment → Environment


Ex: home → car

Ex: playground → classroom


3. Sensory → Sensory


Ex: quiet → loud

Ex: dark → bright


4. Cognitive → Cognitive


Ex: math → reading

Ex: story → instructions


5. Emotional → Emotional


Ex: calm → uncertainty

Ex: uncertain → confidence


EchoAuth adapts to the transition type automatically.


──────────────────────────────────


⭐ 19.5 — TMS RESPONSE RULES (NI–AI Core Logic)


Rule 1 — Announce Every Transition


No surprises.


Rule 2 — Ask Permission for Cognitive Transitions


“Is it okay if we switch?”


Rule 3 — Slow Down During Emotional Transitions


Pacing resets the nervous system.


Rule 4 — Minimize Language During Switch Phase


Too much input = overload.


Rule 5 — Validate After Transition


“You did a good job.”


Validation is healing for ASD nervous systems.


──────────────────────────────────


⭐ 19.6 — SENSORY LOAD ADJUSTMENT (TMS + SECM Integration)


EchoAuth uses SECM (Page 15) to adjust transitions:


Sensory State Adjustments

Regulated normal transition timing

Light Noise double prep time

Uncertain triple prep time

Overstimulated postpone transition

Shutdown Risk DO NOT transition


This prevents accidental meltdowns.


──────────────────────────────────


⭐ 19.7 — EXAMPLES OF CORRECT & INCORRECT TRANSITIONS


Correct Example:


“In one minute, we’ll switch.

Step 1: pause.

Step 2: breathe.

Step 3: we switch together.”


Incorrect Example:


❌ “Okay, time to stop! Let’s go.”

This causes instant overload.

Caregivers receive real-time instructions to reduce tension.
