PHASE 2 — Passive Observation

• Log dates, not emotions

• No speculative planning

Output:

PHASE 3 — Signal Classification


When a signal arrives (email, update, request):


Classify it as:

• 🟢 Informational (receipt, confirmation)

• 🟡 Clarification (minor question)

• 🔴 Actionable (revision, resubmission, interview)


Rule:

No response until signal is classified.


⸻


PHASE 4 — Controlled Response Gate (CEG Interaction)


Only actionable signals pass through:

• CEG enforces:

• Turn order

• Scope limits

• Response timing


Examples:

• NIH requests clarification → respond within scope only

• NSF silence → no action


Output:

✔ Response memo

✔ Logged under submission ID

PHASE 5 — Outcome Resolution


Outcomes:

• ✅ Award

• ❌ Decline


Each outcome routes differently:

• Award → SET 5 (Execution / Build)

• Revise → SET 3 (Targeted re-entry)

• Decline → Archive + wait (no immediate pivot)
