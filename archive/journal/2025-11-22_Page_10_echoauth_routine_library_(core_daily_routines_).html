⭐ 10.1 — CALMING ROUTINE (Signature Routine)


Purpose: Reduce sensory overload and stabilize nervous system.

Length: 3–4 steps

Activation: Safe Mode / Overload / Nighttime / Stress detection


Steps

1. Sit still (or place feet flat)

2. Slow breath in

3. Slow breath out

4. Stay here for a moment


NI–AI Script


“Let’s calm together.

Step 1: Put both feet on the floor.

Step 2: Take a slow breath in…

Step 3: Let it out gently.

Step 4: Stay here. I’m right with you.”


Safe Mode Variant


“Slow breath.

One more.

You’re safe.”


Caregiver Notes

• Gentle grounding

• No emotional probing

• Perfect for meltdowns or overwhelm


────────────────────────


⭐ 10.2 — MORNING START ROUTINE


Purpose: Transition from sleep → readiness.

Length: 3–5 steps

Activation: Morning / caregiver request / user preference


Steps

1. Wash face

2. Brush teeth

3. Get dressed

4. Put on shoes

5. Ready for the day


NI–AI Script


“Let’s get ready together.

Step 1: Wash your face.

Step 2: Brush teeth.

Step 3: Get dressed.

Step 4: Shoes on.

Step 5: You’re ready.”


Safe Mode Variant


“One step: brush teeth.

Next step: get dressed.

We go slow.”


Caregiver Notes

• Reduces morning anxiety

• Predictability improves mood


────────────────────────


⭐ 10.3 — HOMEWORK / STUDY ROUTINE


Purpose: Build focus and reduce task anxiety.

Length: 3–4 steps

Activation: After school / triggered manually


Steps

1. Open notebook

2. Set 5-minute timer

3. Write first sentence

4. Take short break


NI–AI Script


“We can do this step by step.

Step 1: Open the notebook.

Step 2: Set a 5-minute timer.

Step 3: Write the first sentence.

Step 4: Take a short break.”


Safe Mode Variant


“One thing: write one sentence.

Pause after.”


Caregiver Notes

• Breaks reduce stress

• Supports ADHD + ASD

• Builds executive function slowly


────────────────────────

Purpose: Smooth transition from home → outside.

Length: 3–4 steps

Activation: Before leaving home


Steps

1. Put on socks

2. Put on shoes

3. Grab jacket

4. Grab bag


NI–AI Script


“Let’s get ready to go.

Step 1: Socks.

Step 2: Shoes.

Step 3: Jacket.

Step 4: Bag.”


Safe Mode Variant


“Shoes first.

Then jacket.”


Caregiver Notes

• Reduces last-minute overwhelm

• Great for school / appointments

⭐ 10.5 — BEDTIME ROUTINE


Purpose: Lower stimulation → prepare body for rest.

Length: 3–5 steps

Activation: Nighttime / caregiver schedule


Steps

1. Change clothes

2. Brush teeth

3. Turn off lights

4. Slow breaths

5. Lay down


NI–AI Script


“Let’s wind down.

Step 1: Change clothes.

Step 2: Brush teeth.

Step 3: Lights off.

Step 4: Slow breaths.

Step 5: Lay down.”


Safe Mode Variant


“Lights off.

Breathe slow.

I’m here.”


Caregiver Notes

• Helps prevent nighttime anxiety

• Builds consistency


────────────────────────


⭐ 10.6 — EMOTIONAL RESET ROUTINE


Purpose: Bring user from frustration → balance.

Length: 3 steps

Activation: Emotional spikes / frustration / crying


Steps

1. Sit or pause

2. One deep breath

3. Name the feeling (optional)


NI–AI Script


“Let’s reset together.

Step 1: Sit still.

Step 2: One deep breath.

Step 3: If you want, tell me how it feels.”


Safe Mode Variant


“Sit.

Slow breath.

That’s enough.”


Caregiver Notes

• Helps avoid meltdowns

• No pressure to talk


────────────────────────


⭐ 10.7 — “TASK STARTER” ROUTINE (ADHD SUPPORT)


Purpose: Lower friction before starting tasks.

Length: 3 steps

Activation: School, chores, homework, writing tasks


Steps

1. Pick one small thing

2. Start timer (1–2 minutes)

3. Begin the first tiny action


NI–AI Script


“We only start small.

Step 1: Pick one tiny piece.

Step 2: Set a 2-minute timer.

Step 3: Begin the first tiny action.”


Safe Mode Variant


“Let’s start very small: one tiny piece.”


Caregiver Notes

• Reduces task paralysis

• Key ADHD feature


────────────────────────


⭐ 10.8 — “TAKE A BREAK” ROUTINE


Purpose: Reduce sensory pressure and stress.

Length: 3–4 steps

Activation: Overwhelm, boredom, emotional fatigue


Steps

1. Step away

2. Slow breath

3. Sip water

4. Reset


NI–AI Script


“Let’s take a break.

Step 1: Step away for a moment.

Step 2: Slow breath.

Step 3: Sip water.

Step 4: Reset.”


Safe Mode Variant


“Pause.

Breathe.

Water.”


Caregiver Notes

• Helps regulate overstimulation

• Must be simple and predictable


────────────────────────

Step 1: Arms up.

Relax.”


Caregiver Notes
