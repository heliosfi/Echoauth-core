403.5 — Internal Mechanics


(a) Continuity-Lock Matrix (CLM)


Freezes all transition pathways except:

• guardian override

• medical emergency

• critical system failure


(b) Drift-Barrier Engine (DBE)


Prevents internal subsystems (F.I. Kernel, TLE, NSM, MERR) from suggesting or initiating transitions.


(c) External Actor Hard Block (EAHB)


Teachers, doctors, admins CANNOT:

• turn off modes

• lower safety level

• override guardian state

• shorten duration


Every attempt is logged and rejected.


(d) Stability Pulse Checker (SPC)


Every 3–12 seconds (variable based on environment), SPC checks:

• sensory noise

• emotional load

• environmental stability

• device signal quality


If instability increases:

• GSCE EXTENDS safe state duration

• notifies guardian

• alerts environment


(e) Safety-Inertia Buffer (SIB)


Even when everything returns to normal, NI-AI waits extra time before allowing exit (unless guardian approves earlier).


This prevents premature returns to normal conditions.


(f) Guardian-Override Sync Layer (GOSL)


Ensures that even across:

• multiple devices,

• multiple environments,

• multiple caregivers,

the system stays consistent with the PRIMARY guardian’s authority.
