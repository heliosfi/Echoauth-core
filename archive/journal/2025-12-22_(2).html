Narrative Key for Diagram (Drop into SSP or Appendix):

• Start: User input enters via secure interface (TLS-protected).

• Core Flow: Progresses through normalization, emotional analysis, coherence enforcement, pattern mapping, authentication, response stabilization, and logging.

• Safety Branches: At analysis and coherence stages, failures trigger escalation (e.g., anomaly alerts, containment per IR-4). Green paths assume success; red branches handle risks like drift or unauthorized access.

• Controls Mapped: Integrates AC-3 (access enforcement), SI-4 (monitoring), IA-2 (MFA via RMS), AU-2 (audit events). Ensures zero-drift safety with human oversight loops.
