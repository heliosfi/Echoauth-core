📌 USE OF FUNDS (Loan Purpose)


All loan funds will be used to complete the minimum viable product (MVP) of EchoAuth — a caregiver–child biometric authentication system designed for non-verbal autistic children.


The $15,000 loan will be allocated as follows:


1. Software Infrastructure & Cloud Hosting — $4,000


To deploy a secure, stable backend for the EchoAuth chamber:

• Cloud compute (API gateway, authentication chamber)

• Encrypted storage for caregiver/child session keys

• Scalable FastAPI + Python backend hosting


Impact:

Enables the real-time authentication required for EchoAuth’s mission.


⸻


2. Biometric & Audio Processing Pipeline (Development) — $3,500


Funding covers:

• Integration of basic camera + audio preprocessing

• Development of low-latency verification routines

• Testing environment for secure child-caregiver matching


Impact:

Creates the functional backbone that verifies identity and safety.


⸻


3. Front-End Interface + Parent Dashboard — $3,000


Complete the first interface for caregivers:

• EchoAuth login

• Alerts dashboard

• Session notifications

• Child safety indicators


Impact:

Gives caregivers a seamless, easy-to-use portal to manage authentication sessions.


⸻


4. MVP User Testing + Hardware (Minimal) — $2,000


For real-world testing with families:

• A test device for running the chamber

• Small stipends for families testing early prototypes

• Safety compliance checks


Impact:

Ensures EchoAuth works in real homes with real children.


⸻


5. Branding, Legal Prep & MVP Launch Materials — $2,500


Covers:

• Logo + brand compliance (already created, now applying use)

• Privacy policy + terms

• Pre-USPTO setup

• Website launch


Impact:

Allows EchoAuth to operate publicly, onboard families, and meet early-stage compliance needs.


⸻


📌 TOTAL: $15,000 (100%) — All Funds Used for EchoAuth MVP Development and Deployment
