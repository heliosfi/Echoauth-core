PAGE 16 — ASD PROCESSING PATHWAYS


“The Internal Flow of an Autistic Nervous System”


──────────────────────────────────


Autistic processing is not “slower.”

It is different — more detailed, more intense, and more literal.


EchoAuth must respect those pathways, not override them.


This page outlines the six stages that sensory, emotional, and cognitive information travels through inside an autistic brain — and how NI–AI must respond at each stage.


──────────────────────────────────


⭐ 16.1 — SENSORY INPUT PATHWAY


How stimuli enter the ASD nervous system.


Autistic sensory systems often receive:

• more raw data

• less automatic filtering

• higher intensity signals

• overlapping channels

• delayed calming


The Sensory Pathway Steps:

1. Raw Sensory Intake

• Sound

• Light

• Texture

• Movement

• Temperature

• Internal body sensations

2. Amplification Layer

Stimuli often become stronger, not weaker.

3. Filtering Gate

NT brains filter automatically.

ASD brains may have low or inconsistent filtering.

4. Meaning Assignment

Sensory input becomes “safe” or “unsafe.”

5. Emotional Trigger Point

If overwhelmed → anxiety spike

If regulated → curiosity or neutrality

6. Cognitive Processing Entry

Only after all above stages does “thinking” start.


NI–AI must never skip these steps.


──────────────────────────────────


⭐ 16.2 — EMOTIONAL INPUT PATHWAY


How emotions enter, peak, and shape cognition.


Autistic emotional processing tends to be:

• more intense

• more literal

• more transparent

• harder to regulate mid-surge

• deeply influenced by sensory load


Stages:

1. Emotional Perception

The child detects a feeling.

2. Intensity Mapping

ASD intensity ranges spike faster.

3. Sensory Link

Emotional discomfort often links to a sensory trigger.

4. Cognitive Access Gate

If emotion is high, thinking becomes limited.

5. Behavioral Expression

May express through:

• quietness

• stimming

• shutdown

• direct statements

• physical movement

6. Regulation Window

Emotion recovers slower and requires predictability.


NI–AI must always track emotion + sensory together.


──────────────────────────────────


⭐ 16.3 — COGNITIVE PROCESSING PATHWAY


How thoughts are organized and understood.


Autistic cognition is:

• highly logical

• pattern-oriented

• detail-focused

• depth-based

• nonlinear during stress


Stages:

1. Detail Intake

ASD minds notice micro-information first.

2. Pattern Construction

Strong pattern recognition → powerful learning potential.

3. Literal Interpretation

No ambiguity → exact meaning only.

4. Context Locking

ASD minds “lock” into the current topic.

5. Transition Resistance Gate

Switching topics can feel like “starting from zero.”

6. Cognitive Fatigue Point

Overload = shutdown, not confusion.


EchoAuth must avoid topic jumps, ambiguity, and long sequences.
