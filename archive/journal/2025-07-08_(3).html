# SniperBot VIX Codex — LIVE Upgrade 🔥


A CUDA-ready, volatility-aware sniper bot that merges real-time market data with intelligent signal filtering.


---


## 🔁 Modules


| File                        | Purpose |

|-----------------------------|---------|

| `main.py`                   | Bot bootstrap, schedule, and VIX-filtered execution |

| `sniper/data_feed.py`       | Live price data via Alpaca or Polygon |

| `sniper/signal_engine.py`   | RSI, EMA, MACD (cuPy or numpy fallback) |

| `sniper/vix_data.py`        | Real-time VIX fetcher (yfinance) |

| `sniper/vix_tracker.py`     | Classifies VIX zone: Fear, Neutral, or Complacency |

| `sniper/execution_engine.py`| Trade execution via broker |

| `sniper/alert_manager.py`   | Telegram + console alerts |

| `dashboard/app.py`          | FastAPI live UI for charts + VIX |
